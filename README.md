# 考研作战中心

27 届考研备考一站式管理工具。单页应用，所有数据存 localStorage。

## 功能

| 模块 | 说明 |
|------|------|
| 每日复盘 | 记录每日学习完成度、各科用时、错题统计、心情、反思 |
| 每周总结 | 自动汇总七天数据，生成完成度柱状图、各科用时、错题趋势 |
| 时间表 | 每日详细时间安排（A/B 两套方案） |
| 阶段计划 | 数学/408/英语/政治四科分阶段计划 |
| 应急方案 | 防崩盘预案 |
| 课程表 | 学校课程安排 |
| 学习记录 | 来自专注工具的番茄钟数据统计 |

## 特色

- **智能预填**：根据最近 14 天数据自动推荐今日完成度和各科用时
- **设置面板**：自定义背景图、毛玻璃效果、卡片透明度
- **Tab 顺序**：自由拖拽调整导航栏排列
- **网站首页**：指定打开时显示的默认页面
- **数据备份**：一键导出/导入全部数据（JSON）

## 使用

直接用浏览器打开 `考研作战中心.html`，所有数据存浏览器 localStorage。
